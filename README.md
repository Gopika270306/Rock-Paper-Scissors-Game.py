# Rock Paper Scissors Game

## Overview

A simple Rock Paper Scissors game developed using Python and Tkinter. The application allows users to play against the computer through an interactive graphical user interface.

## Features

* Graphical User Interface (GUI)
* Random computer choice generation
* Win, Lose, and Tie result detection
* Interactive gameplay
* Beginner-friendly Python project

## Technologies Used

* Python
* Tkinter

## Run the Application

```bash
python rock_paper_scissors.py
```

## How to Play

1. Click Rock, Paper, or Scissors.
2. The computer randomly selects its choice.
3. The result is displayed on the screen.
